## Look at the project structure
- .NET standard library (MainPage.xaml, App.xaml.cs)
- Android (MainActivity)
- iOS (Main.cs, AppDelegate.cs)
- UWP (MainPage.xaml, App.xml)

## Run the application
- Run on emulator (Android)
- Run on phone
- Use Xamarin Live Player
- Run on Windows (UWP)
